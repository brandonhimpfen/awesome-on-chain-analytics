## Analytics Platforms

Platforms providing dashboards, metrics, and insights into blockchain activity.

- [Glassnode](https://glassnode.com/) — On-chain analytics platform with metrics on flows, supply, and network activity.
- [Nansen](https://www.nansen.ai/) — Wallet labeling and smart money tracking for on-chain insights.
- [IntoTheBlock](https://www.intotheblock.com/) — AI-driven analytics platform for crypto markets.
- [Santiment](https://santiment.net/) — Data platform combining on-chain, social, and development metrics.
- [Token Terminal](https://tokenterminal.com/) — Financial analytics for crypto protocols and networks.

## Blockchain Explorers

Tools for exploring transactions, blocks, and wallet activity.

- [Etherscan](https://etherscan.io/) — Explorer for Ethereum transactions, contracts, and addresses.
- [Blockchain.com Explorer](https://www.blockchain.com/explorer) — Explorer for Bitcoin and other major chains.
- [Blockchair](https://blockchair.com/) — Multi-chain blockchain explorer with advanced search features.
- [Solscan](https://solscan.io/) — Explorer for the Solana blockchain.
- [BscScan](https://bscscan.com/) — Explorer for Binance Smart Chain.

## Query & Data Tools

Tools for querying, analyzing, and visualizing blockchain data.

- [Dune](https://dune.com/) — SQL-based analytics platform for blockchain data dashboards.
- [Flipside](https://flipsidecrypto.xyz/) — Data platform for querying and analyzing blockchain activity.
- [Footprint Analytics](https://www.footprint.network/) — Visual analytics platform for DeFi and Web3 data.
- [Allium](https://www.allium.so/) — Blockchain data platform for developers and analysts.
- [SubQuery](https://subquery.network/) — Data indexing framework for blockchain applications.

## APIs & Data Infrastructure

APIs and infrastructure for accessing and processing blockchain data.

- [Alchemy](https://www.alchemy.com/) — Blockchain developer platform with APIs and data services.
- [Infura](https://infura.io/) — Infrastructure provider for accessing Ethereum and IPFS.
- [QuickNode](https://www.quicknode.com/) — Blockchain API provider for multiple networks.
- [Covalent](https://www.covalenthq.com/) — Unified API for multi-chain data access.
- [Moralis](https://moralis.io/) — Backend platform for Web3 applications with real-time data.

## Wallet Intelligence & Tracking

Tools for analyzing wallet behavior and identifying patterns.

- [Arkham](https://arkhamintelligence.com/) — Intelligence platform linking wallets to entities.
- [Breadcrumbs](https://www.breadcrumbs.app/) — Visualization tool for tracking transaction flows.
- [DeBank](https://debank.com/) — Wallet tracking and DeFi portfolio analytics.
- [Zapper](https://zapper.xyz/) — Dashboard for tracking DeFi positions and wallet activity.
- [Zerion](https://zerion.io/) — Portfolio tracker for on-chain assets and activity.

## DeFi Analytics

Tools focused on decentralized finance metrics and protocols.

- [DefiLlama](https://defillama.com/) — TVL tracking and analytics for DeFi protocols.
- [DeFi Pulse](https://defipulse.com/) — Analytics platform for DeFi market activity.
- [DappRadar](https://dappradar.com/) — Analytics for decentralized applications and usage.
- [TokenInsight](https://tokeninsight.com/) — Market intelligence for crypto and DeFi.
- [APY.Vision](https://apy.vision/) — Analytics for liquidity providers and yield farming.

## Forensics & Compliance

Tools for blockchain investigation, risk assessment, and compliance.

- [Chainalysis](https://www.chainalysis.com/) — Blockchain analysis platform for compliance and investigations.
- [Elliptic](https://www.elliptic.co/) — Crypto risk management and compliance analytics.
- [TRM Labs](https://www.trmlabs.com/) — Blockchain intelligence platform for fraud detection and compliance.
- [CipherTrace](https://ciphertrace.com/) — Crypto intelligence and anti-money laundering solutions.
- [Scorechain](https://www.scorechain.com/) — Risk assessment tools for blockchain transactions.

Sources of blockchain data for research and analysis.

- [CryptoCompare](https://www.cryptocompare.com/) — Data and analytics for crypto markets.
